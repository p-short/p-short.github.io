Ring Modulator

Simple Ring Modulator plugin written in C++ Using the JUCE framework.

This project came about by wanting to design a simple plugin interface to replicate older style guitar pedal stompboxes. I treated this project as a way to futher expand on my knowledge of the JUCE look and feel classes so I could be able to turn the defult GUI components into something more personalised. The ring mod audio effect is created by multiplying the incoming signal by a sinewave with a variable frequency between 20 - 4000 Hz, with a bypass switch to alternate between the dry and processed output signal.

LFO Builder with Scope

Create a unique modulation source by blending the output of two variable wavetable oscillators. Written in C++ Using the JUCE framework.

This is an ongoing project so far, I’ve developed all the necessary GUI components to choose the frequency/shape of the two oscillators, shapes available include sine, triangle, sawtooth, square and perlin noise. The sum of their outputs are used to produce an interesting nonlinear source of modulation. I’m hoping to the extend this by using the summed LFO to control the parameters of a low-pass filter or a flanger.

Midibrot

Animated generative MIDI sequencer audio plugin. Written in C++ using the JUCE framework

Select a musical scale from the Midibrots UI, each interval in that scale is represented by a node circle. The node circles are housed inside and cannot cross the larger boundry circle. The positions of the node circles are determined by a mathematical model based on the Mandelbrot set and can be moved via the position sliders or by modulating the positions using sine wave, cosine wave or Perlin noise. When the rotating arm collides with a node circle the Midibrot will output the MIDI note associated with that particular node circle.

Swarmz

Interactive audio-visual tool for creating generative micro tonal clusters. Written in C++ using openFrame Work and Maximilian libaries

Four autonomous agents move around each other in clusters. Each agent is a sawtooth oscillator, its position on the x axis is used to control its placement within the stereo field and the cutoff frequency of its own lowpass filter. The agents position on the y axis is used to modulate the pitch of the sawtooths frequency and the resonance of its lowpass filter. The radius of the swarm can be manipulated, along with the speed of each agent and the overall amount of pitch modulation, filter cutoff and filter resonance.

Random Walks

Experimental Interactive audio-visual piece

Within this piece Perlin noise and sinusoidal waves are used to modulate aspects of the visuals and sound, these waveforms are used to modulate the level of green and blue in the pieces background and are also (using FM synthesis) used to modulate the modulation frequency of a lone sine wave at 440Hz. The carrier and modulation frequency share a harmonic relationship so sweeping harmonics are added to the fundamental frequency over time.

Click the screen to unleash sprawling visuals, each click also triggers randomly selected notes from an A Minor scale.

Gravitysynth

Interactive audio-visual monophonic synthesiser

A ball is rendered to the screen at a random position on every unit of time specified by the "spawn rate" slider. The probability of a ball being spawned can also be changed using the "Spawn Probability" slider. The balls will fall to the ground simulating gravity, every time a ball bounces a note from the selected scale will be triggered, the number of intervals in a scale is divided by the width of the screen so depending on where the ball lands it will trigger the relevant note in the scale. You can also manipulate the octave, portamento and add and remove wind to the simulation.
